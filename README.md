# Online-Quiz-System
The Online Quiz System is a Python-based application integrated with SQL that allows users to participate in interactive multiple-choice quizzes. The system presents questions from various categories such as Python, SQL, and General Knowledge, evaluates user responses, and calculates scores instantly. It uses a database to store and manage quiz questions efficiently, ensuring scalability and structured data handling. This project demonstrates core concepts of programming, database connectivity, and logical problem-solving while providing a simple and user-friendly quiz experience.
